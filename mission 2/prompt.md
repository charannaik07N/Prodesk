## 🛠️ Debugging Prompts Used

### 💬 Chat UI Issue
**Prompt:**
> My code data is storing in local storage, but when I refresh the page, the data is not showing in the UI. However, the data is still present in local storage. Please help me display the data in the UI after refreshing the page.

**Context:**
> Could you please provide more details about the error you're experiencing in the chat UI? This will help me assist you better.

---

### 📄 PDF UI Issue
**Prompt:**
> I have an error in the PDF UI. Could you please help me identify and resolve the issue when generating or viewing PDFs?

**Context:**
> Could you please provide more details about the error you're experiencing in the PDF UI? This will help me assist you better. Are you encountering any specific error messages or issues when trying to view or generate PDFs?
